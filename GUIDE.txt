TROUBLESHOOTING
â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”â”

Problem: "No data found"
  Solution: Run prepare_kaggle_data.py or collect_data.py first

Problem: "Kaggle API not found"
  Solution: pip install kaggle, then set up API key

Problem: "Webcam not detected"
  Solution: Check camera permissions, try different device index

Problem: "Out of memory"
  Solution: Reduce BATCH_SIZE in train.py (change 32 to 16 or 8)

Problem: "Model accuracy is very low"
  Solution: 
    - Use more training data (collect more samples)
    - Train for more epochs (increase EPOCHS)
    - Lower LEARNING_RATE for more stable training
